# E-Ration Shop

Full-stack ration distribution system with:

- React + Vite frontend
- Node.js + Express backend
- MongoDB database

## Frontend

Install dependencies and start the frontend:

```sh
npm install
npm run dev
```

The frontend runs on `http://127.0.0.1:8080/`.

## Backend

Go to the backend folder, install dependencies, and start the server:

```sh
cd backend
npm install
npm run dev
```

The backend is expected on `http://127.0.0.1:5001/`.

## Main Features

- Citizen, dealer, and admin portals
- Registry-based citizen verification
- Dealer allocation and stock release workflows
- Distribution recording and audit trail
- Central monthly stock tracking
- Notifications and collection-code flow

## Build

Frontend:

```sh
npm run build
```

Backend:

```sh
cd backend
npm run build
```
